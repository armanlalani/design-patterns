# Course: Master Design Patterns in Java

Welcome to the design patterns course workspace! 
Each chapter focuses on a different category of design patterns. Within each category, patterns are explored via progressive modules (`m01`, `m02`, etc.) to demonstrate the evolution from anti-patterns to elegant solutions.

## Structure

* **`src/main/java/com/course/designpatterns/`** - The root namespace for our course.
  * **`ch01_builder_pattern/`** - Discover how to construct complex objects securely.
    * `m01_copy_constructor`
    * `m02_setters_pattern`
    * `m03_builder_pattern`
  * **`ch02_factory_method_pattern/`** - (Coming soon)
  * **`ch03_singleton_pattern/`** - (Coming soon)
